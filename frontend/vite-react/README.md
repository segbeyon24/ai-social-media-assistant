# Frontend (Vite React)

This folder contains the Vite + React + TypeScript frontend for LeanSocial.

Quick start:

1. cd frontend/vite-react
2. npm install
3. Create a `.env` file with `VITE_API_BASE_URL`, `VITE_SUPABASE_URL`, and `VITE_SUPABASE_ANON_KEY`.
4. npm run dev

Notes:
- The `Login` button redirects to the backend auth start endpoint at `VITE_API_BASE_URL`.
- This scaffold is minimal: add chart libraries and richer components as needed.
